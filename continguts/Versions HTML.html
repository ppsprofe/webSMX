Versions HTML

HTML 1

La primera versió d'HTML, coneguda com a HTML 1.0 Strict, es va publicar el 1993. En aquell moment, la creació de llocs web encara estava a les beceroles i el llenguatge era força limitant. Podríeu fer poc més que mostrar text simple a la web utilitzant HTML 1.0. No obstant això, aquest llançament inicial va marcar l'inici d'un viatge significatiu per al desenvolupament web.

1995
HTML 2

El 1995 es publica l' estàndard HTML 2.0. Malgrat el seu nom, HTML 2.0 és el primer estàndard oficial d'HTML, és a dir, l'HTML 1.0 no va existir com a estàndard. HTML 2.0 no suportava taules. Se simplificava al màxim l' estructura del document per agilitar la seva edició, on la declaració explícita dels elements body, html i head és opcional.

 1995
HTML 3

La versió HTML 3.2 es va publicar el 1997 i és la primera recomanació d'HTML publicada pel W3C (Consorci internacional). Aquesta revisió va incorporar els últims avenços de les pàgines web desenvolupades fins al 1996, com applets de Java i text que flueix al voltant de les imatges.

 1997
HTML 4

L' última especificació oficial d' HTML es va publicar el desembre de 1999 i es denomina HTML 4.01. Des de la publicació d' HTML 4.01, el W3C es va centrar en el desenvolupament de l' estàndard XHTML. Per aquest motiu, l'any 2004, les empreses Apple, Mozilla i Opera van mostrar la seva preocupació per la falta d'interès del W3C en HTML i van decidir organitzar-se en una nova associació anomenada WHATWG (Web Hypertext Application Technology Working Group) que va començar el desenvolupament de l'HTML 5.

 1999
HTML 5

El consorci internacional W3C, després d' una evolució de diversos anys, va alliberar l' HTML 5 com a estàndard oficial a finals d' octubre de 2014. HTML 5 incorpora nous elements no contemplants en HTML 4.01. Hi ha diversos canvis respecte a HTML 4.01. Hi ha noves etiquetes, s'introdueix la possibilitat d'introduir àudio i vídeo de forma directa al web sense necessitat de plugins o complements als navegadors, i altres novetats. El W3C anirà llançant progressivament noves evolucions de l'HTML 5.
